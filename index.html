Основные React хуки

  • useState

    Отвечает за изменение состояния, каких-либо элементов внутри компонента. Например, изменение значения счетчика при нажатии на кнопку

    Возвращает массив из 2-х значений:

    1. Значение переменной
    2. Функцию, которая ее изменяет

    В useState(X) указывается начальное значение - X

    const [count, setCount] = useState(0);

  • useEffect

    Необходим для реализации сайдэффектов (внешних эффектов), т. е. сихронизации компонента с "внешним миром", например с сервером

    Принимает 2 параметра:

    • Колбек-функцию (то что нужно сделать)
    • Массив зависимостей. Опционально (при изменении каких данных, повторно вызывать useEffect)

    Далее есть 3 варианта:

    • Если массив присутсвует - useEffect вызывается при каждом изменении массива
    • Если присутсвует пустой массив - useEffect вызывается один раз, при отрисовке компонента
    • Если второго параметра нет, useEffect вызывается при каждом рендере (при каждой перересовке компонента)

    useEffect(() => { //действие, которое нужно выполнить }, [массив зависимостей]);

  • useRef

    В основном используется в React для получения доступа к DOM элементам

    useRef возвращает изменяемый ref-объект, свойство .current которого инициализируется переданным аргументом (initialValue). Возвращённый объект будет сохраняться в течение всего времени жизни компонента.

    const inputEl = useRef(null); // `current` указывает на смонтированный элемент `input` inputEl.current.focus();
